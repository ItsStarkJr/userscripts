Installing a script is quite simple, follow the steps below.

1. Install Tampermonkey if you haven't already.

2. In Tampermonkey, go to Utilities and paste the link to the script in Import from URL text area. Do not forget to press Install

3. That's it. Now if you want to keep up to date with my very active updating schedule, make sure you set Userscript Update > Check Interval in your Tampermonkey settings to anything but Never, and it will take care of that for you.

4. If at any point something doesn't work as it should, or you understandably just want to tell me how horrendous my code is, do let me know.
