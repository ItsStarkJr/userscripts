puzz.link - Add direct links

Adds direct links to all puzzles on the current page. To be opened by clicking the thumbnail.

https://gist.github.com/ItsStarkJr/401f2abafb12e938da79f8c83759be58/raw/add_direct_links.user.js

puzz.link - Add open all button

Adds button to puzz.link database to open all puzzles on the current page. Currently not working without the 'add direct links' script. You may need to grant puzz.link permission to open pop ups.

https://gist.github.com/ItsStarkJr/1817bcf03146656c6fa976afbf2718e2/raw/add_open_all_button.user.js

puzz.link - Database styling

Major re-styling of the puzz.link database.

https://gist.github.com/ItsStarkJr/8ade2ee8994b3c8ea3c84e0947338cfe/raw/puzzlink_db_css.user.js

Installing a script is quite simple, follow the steps below.

1. Install Tampermonkey if you haven't already.

2. In Tampermonkey, go to Utilities and paste the link to the script in Import from URL text area. Do not forget to press Install.

3. That's it. Now if you want to keep up to date with my very active updating schedule, make sure you set Userscript Update > Check Interval in your Tampermonkey settings to anything but Never, and it will take care of that for you.

4. If at any point something doesn't work as it should, or you understandably just want to tell me how horrendous my code is, do let me know.
